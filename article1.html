قسم: استخراج الكنوز المدفونة

إحياء تطبيق "Termux" - وحش البرمجة المنسي

مرحباً بكم في مدونة خالد اللعيب. اليوم ننبش في "قبر جوجل بلاي" لنستخرج واحداً من أقوى الأدوات التي عرفها تاريخ الأندرويد، وهو تطبيق Termux. هذا التطبيق ليس مجرد برنامج عادي، بل هو بيئة محاكي Linux كاملة تعمل على هاتفك الجوال دون الحاجة لعمل "روت".

لماذا نعتبره كنزاً؟

في الآونة الأخيرة، قامت جوجل بفرض قيود صارمة على متجرها، مما جعل نسخة المتجر من هذا التطبيق "ميتة" ولا تتلقى تحديثات. لكن المبرمجين الحقيقيين يعرفون أن الروح ما زالت تدب في هذا التطبيق عبر المصادر المفتوحة. يمكنك استخدامه لبرمجة سكريبتات JavaScript، Python، وحتى اختراق الثغرات الأخلاقية وتجربة الحماية.

كود الأوامر الأساسي (JS Logic)

إليك كيف يمكنك استدعاء أوامر النظام عبر سكريبت بسيط داخل البيئة المفتوحة:

// JavaScript Node.js Script for Termux API const { exec } = require('child_process'); exec('termux-vibrate -d 500', (err, stdout, stderr) => { if (err) { console.error('فشل الوصول للنظام المفتوح'); return; } console.log('تم تفعيل الهزاز عبر كود JS بنجاح!'); });
